# badged-lend
The pneumonia risk model for hemorrhagic stroke is named BADGED-3H (Brain hernia, Admission record, Decubitus ulcer, Gastrointestinal bleeding, Endotracheal intubation, Dysphagia, Hydrocephalus, Hyperleukocytosis, Hypertension), emphasizing the severity and complexity of pneumonia to remind doctors to be attentive during diagnosis and treatment. The pneumonia risk model for ischemic stroke is named LEND-CARE (Lung disease, Endotracheal intubation, Number of resuscitations, Dysphagia, Decubitus ulcer, Age, and Admission record), highlighting key aspects in elderly care and the importance of attentiveness.
